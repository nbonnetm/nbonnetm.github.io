Industrial Design · Fabrication

"Curioso" Belt Buckle

Year 2022
Role Designer · Fabricator
Materials Bronze, Rhodium Plating, Brass Rivets
Tools SolidWorks, FDM 3D Printing, Silicone Molding, Wax Injection

Riffing off of an existing buckle mechanism while implementing a visually intruguing form. Facial form was derived from the functional form of a "Cobra" style buckle mechanism. This project serves as an initial prototype for a design intended for larger-scale production. This version was produced using cast metal.
